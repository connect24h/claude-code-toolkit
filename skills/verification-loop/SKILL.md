# Verification Loop スキル

コード変更後に包括的な検証を実行します。

## トリガー

以下のキーワードが含まれる場合に自動適用:
- 実装完了
- コミット前
- PR前
- デプロイ前

## ワークフロー

```
┌─────────────────┐
│    型チェック    │ tsc --noEmit
└────────┬────────┘
         ↓ パス
┌─────────────────┐
│     リント      │ eslint
└────────┬────────┘
         ↓ パス
┌─────────────────┐
│    テスト       │ npm test
└────────┬────────┘
         ↓ パス
┌─────────────────┐
│    ビルド       │ npm run build
└────────┬────────┘
         ↓ パス
      ✅ 完了
```

失敗した場合は修正して再実行。

## コマンド

```bash
# 型チェック
npx tsc --noEmit

# リント
npm run lint

# テスト
npm test

# ビルド
npm run build

# 全て実行
npm run lint && npm test && npm run build
```

## チェックリスト

- [ ] 型チェックパス
- [ ] リントパス（警告なし）
- [ ] テストパス（カバレッジ80%+）
- [ ] ビルド成功

## 関連コマンド

- `/verify`: 検証を明示的に開始
- `/build-fix`: ビルドエラーを修正
