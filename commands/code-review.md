# /code-review コマンド

code-reviewerエージェントを呼び出して、コードレビューを実行します。

## このコマンドがすること

1. **正確性チェック** - ロジックの正しさ
2. **セキュリティチェック** - 脆弱性の検出
3. **パフォーマンスチェック** - 効率性の評価
4. **保守性チェック** - 読みやすさ、構造
5. **テストチェック** - カバレッジと質

## いつ使うか

- 実装完了後
- PRを作成する前
- 他の人のコードをレビュー
- リファクタリング後

## 使用方法

```
/code-review [対象ファイル/機能]
```

## 例

```
/code-review src/services/auth.ts
/code-review 最近の変更
```

## レビュー結果

```markdown
# コードレビュー

## 良い点
- [良い点]

## 改善が必要な点

### 🔴 クリティカル（修正必須）
- [問題]

### 🟡 重要（推奨）
- [問題]

### 🟢 軽微（任意）
- [問題]

## 結論
[承認/要修正/却下]
```
