# /verify コマンド

verification-loopスキルを使用して、コードを包括的に検証します。

## このコマンドがすること

1. **型チェック** - `tsc --noEmit`
2. **リント** - ESLint
3. **テスト** - 全テスト実行
4. **ビルド** - 本番ビルド確認

## 検証ループ

```
型チェック → リント → テスト → ビルド
     ↑                              ↓
     └─── 失敗したら修正して再実行 ←─┘
```

## いつ使うか

- 実装完了後
- PRを作成する前
- コミット前
- デプロイ前

## 使用方法

```
/verify
```

## 実行内容

```bash
# 型チェック
npx tsc --noEmit

# リント
npm run lint

# テスト
npm test

# ビルド
npm run build
```

## 出力例

```markdown

## 型チェック
✅ パス

## リント
⚠️ 2件の警告

## テスト

カバレッジ: 85%

## ビルド
✅ 成功

## 結論
✅ 検証完了（警告あり）
```
